1. **Explore** — Browse the [Project Catalog](#-project-catalog) above to find projects that interest you
2. **Try it** — Check out our [Jupyter Notebooks](https://github.com/agstack/jupyter-notebooks) for hands-on API examples
3. **Deploy** — Use [OpenAgri Bootstrap](https://github.com/agstack/OpenAgri-Bootstrap-Deployment) to spin up the full OpenAgri platform
4. **Contribute** — Look for repos tagged `hacktoberfest` or browse [open issues across the org](https://github.com/issues?q=is%3Aopen+is%3Aissue+org%3Aagstack)
5. **Connect** — Visit [agstack.org](https://agstack.org) or join our [meetings](https://github.com/agstack/meetings)
